# 🎨 Interactive Web Assignments 🌐  

A collection of interactive web assignments, including a **Profile Card, Anagram Checker, Employee Management System, To-Do List, and Image Slider**. Built with **HTML, CSS, and JavaScript** using Bootstrap for styling.  

## 🚀 Features  
✔ **Profile Card** – Hover effects, animations, and social media icons.  
✔ **Anagram Checker** – Checks if two words are anagrams.  
✔ **Employee Management System** – Displays employee details dynamically.  
✔ **To-Do List** – Add and remove tasks interactively.  
✔ **Image Slider** – Browse through images with next/previous buttons.
